Author: Dylan Chung

Description:
	A Java implementation of the Isolation game using minimax with alpha beta pruning and iterative deepening. 

How to Run:
	1) Open up Command Prompt 
	2) Enter the following commands:
		cd (filepath of folder where you saved code)
		javac Board.java Driver.java Minimax.java Node.java Player.java UserInterface.java
		java Driver
	3) Follow the Prompts on screen
